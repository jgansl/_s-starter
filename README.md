### Live Server - Hot Reload

#### MacOS

- run `npm install` & then remove HTTPS_KEY, HTTPS_CERT from .env.local (generated from npm install)

#### Window

- run `npm install` & then update HTTPS_KEY, HTTPS_CERT in .env.local
